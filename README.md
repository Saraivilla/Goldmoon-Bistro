📌 GoldMoon Bistro

Aplicación Android desarrollada en Java orientada al sector gastronómico. Permite simular un flujo básico de gestión de restaurante, incluyendo inicio de sesión, registro de información del negocio y visualización de datos.

🚀 Funcionalidades: 
- Login simulado (sin base de datos)
- Registro de restaurante y rubro
- Visualización de la información ingresada
- ListView con nombres de productos
  
📱 Estructura de la app
- Activity 1: Login simulado
- Activity 2: Registro del restaurante
- Activity 3: Visualización de datos
- Activity 4: Listado de productos

🛠️ Tecnologías
- Java
- Android Studio
- XML (layouts)
